# 文章初始化说明

## 使用方法

1. 在微信开发者工具中，右键点击 `cloudfunctions/initArticles` 文件夹
2. 选择"上传并部署：云端安装依赖"
3. 部署完成后，在云开发控制台中手动调用该函数一次
4. 函数会自动向 `articles` 集合插入30篇文章数据

## 文章分类

- **科学戒烟 (scientific)**: 5篇
  - 尼古丁替代疗法
  - 戒烟药物治疗
  - 联合药物疗法
  - 戒烟门诊
  - 戒断症状管理

- **心理调节 (psychology)**: 5篇
  - 克服心理依赖
  - 正念冥想
  - 建立戒烟信心
  - 应对情绪波动
  - 社交压力应对

- **生活习惯 (lifestyle)**: 5篇
  - 健康饮食指南
  - 运动锻炼
  - 改善睡眠质量
  - 建立无烟环境
  - 压力管理

- **应对技巧 (coping)**: 10篇
  - 三分钟法则
  - 替代行为
  - 应对复吸
  - 利用戒烟APP
  - 家人支持
  - 戒烟日记
  - 奖励自己
  - 戒烟互助小组
  - 可视化进展
  - 长期维持策略

## 注意事项

- 该云函数只需执行一次
- 执行前请确保 `articles` 集合已创建
- 如需重新初始化，请先清空集合
- 文章数据包含标题、分类、摘要、内容、标签等完整信息
