## 🔄 登录流程变化

### 之前（云函数模式）
```
小程序启动
  → wx.cloud.init()
  → wx.cloud.callFunction('login')
  → 云函数自动获取openid
  → 返回用户数据
```

### 现在（纯API模式）
```
小程序启动
  → wx.login() 获取code
  → POST /api/auth/wxlogin (code)
  → 后端调用微信接口换取openid
  → 返回openid和token
  → POST /api/login (openid)
  → 返回用户数据
```
